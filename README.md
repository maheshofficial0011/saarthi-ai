# Saarthi AI

**Saarthi AI** is a personal agentic AI assistant designed to act as a digital companion for learning, coding, planning, memory, automation, file understanding, and voice-based interaction.

It is powered by the **Manas Engine** and a modular system of **Saarthi Agents**.
