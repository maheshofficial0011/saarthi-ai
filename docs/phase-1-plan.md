# Phase 1 Plan — Saarthi AI Foundation

## Goal

Build the first working version of Saarthi AI with authentication, dashboard, chat interface, and basic AI response flow.

---

## Features

- Landing page
- Login page
- Signup page
- Protected dashboard
- Chat page
- Sidebar
- Message bubbles
- Settings page
- Basic assistant identity
- Conversation saving structure

---

## Not Included in Phase 1

- Voice
- File upload
- Desktop control
- Advanced automation
- Multi-agent execution
- Long-term memory engine
